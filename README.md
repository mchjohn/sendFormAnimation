## Sobre esse projeto

Animação com React Native, Expo, TypeScript e Lottie.

## Instalação

```
$ git clone git@github.com:mchjohn/sendFormAnimation.git

$ cd sendFormAnimation
```

**Instale as dependências**

```
$ yarn
```

**Rodando a aplicação**

```
$ expo start
```
